# 📊 E-Commerce Sales Analysis Dashboard

## 📌 Objective
To analyze sales, profit, and customer behavior using an interactive Power BI dashboard.

## 🛠 Tools Used
- Power BI  
- Excel  
- DAX  

## 📈 Key Insights
- Maharashtra has the highest sales contribution  
- Clothing category contributes the maximum quantity  
- COD is the most preferred payment mode  
- Profit shows a decline in mid-year months  

## 📊 Features
- Interactive dashboard with slicers (Category, Quarter)  
- KPI cards for Total Sales, Profit, Quantity, and AOV  
- Visual analysis by state, category, and sub-category  
- Monthly profit trend analysis
